All methods

  1. [Set:{variable}:{value}] -- Sets the variable to the amount
  2. [Draw_Move:{Point X}, {Point Y}] -- Moves the beginning of a line to the target points
  3. [Draw_Line:{Point X}, {Point Y}] -- Draws a line ending in the points, and starting in the moved position(Draw_Move)
  4. [Draw_Color:{R},{G},{B},{A}] -- Sets the render color (rgba format)
  5. [Draw_Render] -- Renders everything
  6. [Draw_Clear] -- Clears the screen
  7. [Load_Image:{"Source"}:{Id}] -- Loads an image from the web, that can be later drawn, and saves an Id for it.
  8. [Draw_Image:{Loaded image ID}, {Loc. X}, {Loc. Y}, {Size X}, {Size Y}] -- Draws an image from the loaded image Id collection
  9. [Play_Audio(/Play_Sound):{"Source"}] -- Plays an audio from the web
  10. [Stop_Audio(/Stop Sound)] -- Stops all audio
  11. [Draw_Rect:{Loc. X}, {Loc. Y}, {Size X}, {Size Y}] -- Draws a scaleable Rectangle
  12. [Draw_FillRect:{Loc. X}, {Loc. Y}, {Size X}, {Size Y}] -- Draws a scaleable Filled Rectangle
  13. [Draw_Circle:{Loc. X}, {Loc. Y}, {Scale}] -- Draws a scaleable Circle
  14. [Draw_FillCircle:{Loc. X}, {Loc. Y}, {Scale} -- Draws a scaleable Filled Circle
  15. [Draw_Text:{"Text"}, {Loc. X}, {Loc. Y}] -- Writes text
  16. [Draw_Data:{Variable}, {Loc. X}, {Loc. Y}] -- Outputs variable data
  17. [If:{Condition}:{{Method}}] -- An if statement, that can excecute a method, if true -- Example: If: 10 > 5: {Alert};
  18. [Alert] -- Debug Alert Option
  19. [Keypress:{Variable}] -- Sets the variable to the last key press
  20. [Mouse_X:{Variable}] -- Sets the variable to the x axis of the mouse
  21. [Mouse_Y:{Variable}] -- Sets the variable to the y axis of the mouse
  22. [LocalMouse_X:{Variable}] -- Sets the variable to the x axis of the mouse local to the canvas
  23. [LocalMouse_Y:{Variable}] -- Sets the variable to the y axis of the mouse local to the canvas
  24. [Event:{"Event Name"}] -- Executes an event by it's name
  25. [Event_Id:{Event ID}] -- Executes an event by it's ID
  26. [Random:{Variable}, {Max} -- Sets the variable to a random number between 0, and Max

Syntax

[Command] [Specification - :] [Parameter - ,] [Seperator - ;]

Maths and variables

While the engine is not perfect, it allows simple mathematics (+, -, *, /) and supports the use of variables. To set a variable, use "Set:{VARIABLE HERE!}:{TARGET VALUE!}". These values can include a singular mathematical expression, such as "...:1 + 1", or "...:var - 5".

Conditions

WEE Engine allows conditions to be used. An if statement is made up like this: "If: {CONDITION, SYNTAX EXPLAINED LATER}: {EVENT TO EXECUTE}". A CONDITION can include ">", "<", "==", or "!=". Simply nested if statements can throw JS errors, so I advise using EVENTS, to build nested if statements.
